\begin{itemize}
  \item Desarrollo de una aplicación web, denominada Proteopathogen (v 1.0)
  respaldada por una base de datos
  para recoger, almacenar y analizar resultados de identificación de 
  proteínas procedentes de experimentos de proteómica
  relacionados con \textit{Candida albicans}
  \item Adopción del formato estándar de identificaciones mzIdentMl 
  como fuente de datos para la base de datos y aplicación web Proteopathogen (v 2.0)
  \item Recopilación de resultados de espectrometría de masas para la  %no resultados de identificaciones, sino los espectros
  creación de un PeptideAtlas, o Atlas Peptidico, de \textit{Candida albicans}
  \item Reanálisis de los datos existentes en el PeptideAtlas original
  usando una nueva base de datos con secuencias específicas de alelo, un
  un procesamiento multi-algoritmo y resultados de nuevos experimentos diseñados \textit{ad hoc}
  para mejorar la cobertura del proteoma.

  %de proteomica dirigida (MRM) empleados para detectar proteinas en el 
  %contexto del consorcio espaol dedicado al mapeo cromosoma-centrico en el proyecto proteoma humano (HPP) 
\end{itemize}
