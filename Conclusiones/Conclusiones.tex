\chapter*{Conclusiones}
%mirar problema con fancyheader y secciones no numeradas, pagina 21-22 del manual
%\fancyhead[RO,LE]{\sc{Conclusiones}}
\addcontentsline{toc}{chapter}{Conclusiones}



\begin{enumerate}

\item La base de datos y aplicación web creada, denominada Proteopathogen,
es una herramienta pública \textit{online} de gran utilidad para la visualización
y análisis de resultados de proteómica en estudios que usan 
\textit{C. albicans} como organismo modelo de hongos patógenos.

\item La adopción del estándar mzIdentML como formato de origen para
incorporar nuevos datos en Proteopathogen asegura la estabilidad y futuro
de este proyecto ya que es posible obtener archivos con resultados
de identificaciones en este formato independientemente del procesamiento experimental
y computacional.

\item Se ha creado un PeptideAtlas de \textit{C. albicans} estableciendo
por primera vez una caracterización a gran escala del proteoma
de un modelo de hongo patógeno en el proyecto global PeptideAtlas.


\item El PeptideAtlas de \textit{C. albicans} describe 71310 péptidos y 4174
proteínas (para un FDR de 0,10\% a nivel de PSM),
supone la caracterización más exahustiva del proteoma de este organismo (66\%)
y es el recurso más completo y fiable disponible públicamente.

\item En el PeptideAtlas de \textit{C. albicans} se describen 2860 proteínas
para las que sus correspondientes ORFs se denominan
\textit{uncharacterized} por carecer de un producto génico conocido, lo que supone un 63\% de éstos.



\end{enumerate}
